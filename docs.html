Admin workflow

Recommended order for a clean rollout.

1

Generate defaults

Boot once, then review config.yml, tool-upgrades.yml, tool-skins.yml, and messages.yml.

2

Test crafting

Craft one normal upgrade and one netherite skin recipe with separate one-item ingredient slots.

3

Inspect and fix

Use /coretoolsx inspect and /coretoolsx fixitem on skinned tools before public rollout.
